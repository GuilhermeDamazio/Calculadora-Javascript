## Descrição do projeto:

O projeto visa a criação de uma calculadora simples que realiza as seguintes operações matemáticas: **adição, subtração, multiplicação e divisão**.

## ✔️ Tecnologias utilizadas:

- ``Javascript``
- ``ReactJS``
- ``Tailwind CSS``
